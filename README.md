# Proxy Configurations

本项目用于开源并备份我的代理配置文件，涵盖 Mihomo (Clash Meta)、OpenClash 与 Loon。

## 目录说明
- **mihomo/**:
    - **desktop&mobile/**: 适用于桌面端与移动端的 Mihomo 配置文件。
    - **OpenClash/**: 针对 OpenWrt OpenClash 优化的配置文件。
- **loon/**:
    - **configs/**: Loon 主配置文件 (`.conf`)。
    - **rules/**: 自定义规则集 (`.list`)。

## 注意事项
- 请在导入配置前，确保已替换为自己的订阅链接或节点信息。
- 推荐配合定时同步脚本使用。
